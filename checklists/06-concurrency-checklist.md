# Concurrency checklist

- [ ] Delovodni broj se ne računa u Canvas aplikaciji.
- [ ] Postoji backend reservation flow/service.
- [ ] Postoji retry-on-conflict.
- [ ] Postoji unique constraint gde je moguće.
- [ ] Postoji audit svake rezervacije.
- [ ] Postoji error queue.
- [ ] Postoji operation/correlation ID.
- [ ] Postoji kontrolisana greška ka korisniku.
