# Prompt: Create Flow Implementation Guide

Create a manual implementation guide for a required Power Automate flow.

Include:
- name
- purpose
- trigger
- inputs
- outputs
- connections
- service account requirement
- SharePoint lists/libraries
- detailed action sequence
- expressions
- OData filters
- REST endpoints
- Parse JSON schemas
- Try/Catch/Finally
- AuditLog
- test cases
- what human must return after implementation
