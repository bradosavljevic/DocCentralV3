# SharePoint Guide: AppConfig Schema

Every AppConfig update must validate JSON before writing.

## Minimum schema coverage

- DelovodneKnjige
- Stanja
- ProcesConfig
- TipoviDokumenta
- VrsteDokumenta
- OrganizacioneJedinice
- Settings
- EfaktureParametri
- Translations
- Users
- Valute
- RokoviCuvanja
- RegistracioneJedinice

## JSON row ID

Each array row should include `ID` as GUID.
