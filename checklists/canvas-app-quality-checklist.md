# Checklist: Canvas App Quality

- [ ] Responsive layout
- [ ] Clean App Checker
- [ ] Delegable formulas
- [ ] No cross-screen control references
- [ ] No unnecessary nested galleries
- [ ] No live search on each character
- [ ] IfError around Patch/Flow calls
- [ ] Clear Notify messages
- [ ] gbl/loc/col naming convention
- [ ] App.Formulas or named formulas where useful
- [ ] Centralized config loading
- [ ] Multi-language support
- [ ] Serbian number/date formatting
- [ ] No hardcoded tenant-specific group IDs
- [ ] No direct SharePoint write for end-user operations
