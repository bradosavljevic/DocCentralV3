## Potvrđeno

Canvas aplikacija učitava App Config u `colMasterData`.

## Identifikovane oblasti

- Delovodne knjige
- Proces config
- Stanja
- Permisije
- Tipovi dokumenata
- Valute
- Vrste dokumenata
- Prevodi
- Podešavanja

## Preporuka

Svaki App Config zapis treba da ima poseban dokument sa JSON schema, ekranima koji ga koriste, flow-ovima koji ga koriste i validation pravilima.
