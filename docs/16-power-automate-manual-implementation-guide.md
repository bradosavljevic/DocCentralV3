# 16 - Power Automate Manual Implementation Guide

MCP Canvas / Claude Code cannot create Power Automate flows directly.

Every required flow must have a manual implementation document.

## Required sections per flow

1. Flow name
2. Purpose
3. Trigger
4. Inputs
5. Outputs
6. Connections
7. Service account requirement
8. SharePoint lists/libraries used
9. Environment variables used
10. Detailed action sequence
11. OData filter queries
12. REST API URIs if used
13. Parse JSON schema
14. Select mappings
15. Conditions
16. Apply to each logic
17. Try/Catch/Finally scopes
18. Retry policy
19. Timeout policy
20. AuditLog write
21. Response to Power Apps
22. Test scenarios
23. What human must return to Claude after implementation
