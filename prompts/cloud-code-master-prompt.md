# Cloud Code Master Prompt — DocCentral

Koristi dokumentaciju iz ovog repozitorijuma kao izvor istine za razvoj nove enterprise verzije DocCentral rešenja.

Najvažnije pravilo:

```text
Nikada ne dozvoli dupli delovodni broj.
```

Nova aplikacija treba da bude responzivna, višejezična, zasnovana na Power Apps Canvas UI sloju i Power Automate backend operacijama.
