# DocCentral GitHub Documentation Pack

Ovaj paket sadrži početnu strukturu za GitHub dokumentaciju, Claude Code uputstva, template fajlove i checkliste.

## Folderi

- `claude-code` – uputstva i pravila za Claude Code
- `templates` – Markdown šabloni za dokumentaciju
- `checklists` – checkliste za analizu i kontrolu kvaliteta

## Kako koristiti

1. Raspakuj ZIP.
2. Kopiraj foldere u root GitHub repozitorijuma.
3. Pokreni:

```bash
git add .
git commit -m "Add Claude Code documentation templates and checklists"
git push
```

## Pravilo

Svi dokumenti treba da budu zasnovani na stvarnim fajlovima i dokazima iz Power Platform solution-a i SharePoint konfiguracije.
Ako nešto nije potvrđeno, koristi oznaku:

`NEPOZNATO`
