# Checklist: Power Automate Quality

- [ ] Clear trigger
- [ ] Clear input schema
- [ ] Controlled output to Power Apps
- [ ] Service account connection
- [ ] Error handling scopes
- [ ] Try/Catch/Finally pattern where needed
- [ ] CorrelationId
- [ ] Audit log write
- [ ] Retry logic for critical operations
- [ ] No duplicate registry number risk
- [ ] Handles reserved number safely
- [ ] Does not delete reserved number before document creation success
- [ ] Handles permission assignment errors
- [ ] Returns user-friendly error
- [ ] Technical error stored in log
