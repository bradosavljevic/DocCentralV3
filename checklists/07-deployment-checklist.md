# Deployment Checklist

## Pre deployment

- [ ] Solution exportovan
- [ ] Connection references proverene
- [ ] Environment variables proverene
- [ ] SharePoint liste kreirane
- [ ] SharePoint biblioteke kreirane
- [ ] Kolone proverene
- [ ] Indeksi provereni
- [ ] Unique constraints provereni
- [ ] Grupe proverene
- [ ] Service account spreman
- [ ] Licence proverene

## Deployment

- [ ] Solution importovan
- [ ] Connection references povezane
- [ ] Environment variables podešene
- [ ] Flow-ovi uključeni
- [ ] Canvas app publishovana
- [ ] Permissions primenjene
- [ ] Test korisnici dodati

## Post deployment

- [ ] Smoke test aplikacije
- [ ] Test kreiranja dokumenta
- [ ] Test izmene dokumenta
- [ ] Test arhiviranja
- [ ] Test approval procesa
- [ ] Test concurrency scenarija
- [ ] Test error handling-a
- [ ] Test audit loga
- [ ] Test pristupa za običnog korisnika
- [ ] Test pristupa za administratora

## Rollback

- [ ] Backup prethodne verzije
- [ ] Plan vraćanja flow-ova
- [ ] Plan vraćanja aplikacije
- [ ] Plan vraćanja SharePoint konfiguracije
- [ ] Kontakt osobe za incident
