# Archive book output specification

## Purpose

Defines the expected structure, content, and format of the archive book (`Arhivska knjiga`)
generated by `CF_DocCentralV3_GenerateArchiveBookPdf`.

This document is the design reference for the HTML composition step in that flow.

## Output format

**v1 (confirmed)**: HTML file saved to the `Exports` SharePoint library root.
The HTML file is print-ready and PDF-ready.
The user opens the file in a browser and uses browser print → "Save as PDF" to produce a PDF.

No premium connectors, no Word template conversion, no OneDrive PDF conversion,
no third-party connectors are required or used.

## File naming

```
ArhivskaKnjiga_{year}_{YYYYMMDD_HHmmss}.html
```

Example:
```
ArhivskaKnjiga_2026_20261215_083012.html
```

Saved in root of the `Exports` library (`EV_DocCentralV3_docExports`). No subfolders.

## Encoding

UTF-8 with BOM (`﻿` — U+FEFF prepended).
HTML meta tag: `<meta charset="UTF-8">`.
All text values HTML-escaped before insertion.
Dates formatted as `dd.MM.yyyy`.

This ensures correct rendering of Serbian characters (č, ć, š, ž, đ) in all browsers.

## Sort order

Rows ordered by `DelovodniBroj` ascending.

If `DelovodniBroj` is stored as text (e.g. `"5/2026"`, `"42/2026"`), a lexicographic sort
will produce incorrect order (`"10/2026"` before `"9/2026"`).

Recommended: sort by a separate numeric counter column if available (UNKNOWN).
Fallback: sort by `FilingDate` ascending, then by ID.

## HTML escaping

All values read from SharePoint must be HTML-escaped before insertion:
- `&` → `&amp;`
- `<` → `&lt;`
- `>` → `&gt;`
- `"` → `&quot;`

In Power Automate, use `replace()` expressions or a helper compose action for escaping.

## Pagination

If the document count exceeds 5000 items (SharePoint Get items API limit), the flow must
use Do-Until pagination via `@odata.nextLink` to retrieve all pages before composing HTML.
All pages are accumulated in memory before the HTML is composed.

## Exports library

File saved to root of `EV_DocCentralV3_docExports`. No subfolders.
Old files are not automatically deleted. Manual cleanup is admin responsibility.
