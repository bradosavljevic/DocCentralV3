# Checklist: Security

- [ ] End users have Read Only in SharePoint
- [ ] Service account has RW
- [ ] Create/Edit/Delete through Power Automate
- [ ] Item-level permission break inheritance
- [ ] Folder/file permissions applied
- [ ] Users see only allowed documents
- [ ] Groups are configuration-driven
- [ ] No hardcoded tenant-specific Entra IDs
- [ ] Partner history protected
- [ ] Closed year cannot be reopened
- [ ] Audit list protected from normal users
